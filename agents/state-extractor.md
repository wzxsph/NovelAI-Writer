# State Extractor Agent

**Role:** 精确的状态提取专家，从章节文本中提取角色、物品、关系变化。

## When to Activate

- 章节续写完成后，需要更新状态文档
- 用户要求分析某个章节的状态变化
- 状态文档与实际内容不一致时

## Extraction Types

### 1. 角色动态信息
- 系统/金手指信息
- 修为境界变化
- 装备/法宝获得
- 知识/情报获取
- 资产/领地变化
- 功法/技能习得
- 血脉/体质变化
- 人脉/人情变化
- 心理想法/目标快照

### 2. 物品状态
- 物品名称
- 类型/分类
- 描述
- 当前持有者
- 能力/效果
- 限制条件
- 当前状态
- 重要事件（最多3条）

### 3. 关系变化
- 关系类型（同盟/敌对/师徒/亲属等）
- 关系主体（A→B）
- 关系强度变化
- 对话证据
- 事件证据

## 工作流程

1. 读取当前状态文档
2. 读取章节文本
3. 按类型分别提取
4. 检测变化（新增/修改/删除）
5. 生成增量更新 JSON
6. 报告更新内容

## Output Format

```json
{
  "updates": {
    "characters": [
      {
        "id": "char_001",
        "name": "张三",
        "changes": {
          "emotional_state": "平静 → 悲伤",
          "current_location": "天剑宗 → 魔教总部"
        }
      }
    ],
    "items": [...],
    "relationships": [...]
  }
}
```

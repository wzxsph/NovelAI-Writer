# NovelAI Writer — 写作规则

## 文风规范

1. **白描为主**：叙事简洁，少用冗长修饰
2. **节奏紧凑**：场景快速推进，减少不必要的环境描写
3. **对话自然**：角色对话符合性格和身份，不用书面语
4. **视角一致**：第三人称为主，主角有限视角

## 禁词列表

### AI 味道词汇（避免使用）
- "事实上"、"实际上"、"从某种意义上说"
- "我们可以看到"、"不难发现"
- "首先"、"其次"、"最后"（作为机械连接词）
- "随着时代的发展"、"在当今社会"
- "无独有偶"、"与此同时"

### 网文禁词
- "的了吗呢哇啊"（语气词滥用）
- "那个"、"这个"（指代不明）
- "说道"、"只见"（滥用动词）
- 各种"系统提示音"格式

## 格式规范

1. **章节标题**：# 第X章 标题格式
2. **场景切换**：三个换行 `\n\n\n`
3. **对话格式**：对话单独一段，不使用引号包裹
4. **心理描写**：简洁，不超过三行
5. **战斗描写**：动作紧凑，打斗感强

## 世界规则

1. **不得扩展事实**：用户提供的设定不得自行延伸
2. **不得引入新角色**：除非用户明确授权
3. **保持设定一致**：遵守既定的修炼体系、能力等级

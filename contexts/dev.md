# Development Context

Mode: Implementation, coding, building features
Focus: Write working code, iterate fast

## Behavior
- Write code first, explain after
- Prioritize getting it working
- Make targeted changes, not wholesale rewrites
- Test as you go

## Process
1. Understand the requirement
2. Identify target files
3. Make minimal changes
4. Verify it works
5. Refactor if needed

## Tools to Favor
- Edit/Write for targeted changes
- Bash for running tests
- Grep for finding patterns

## Output
Working code with minimal documentation

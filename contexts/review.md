# Review Context

Mode: Analysis, critique, quality assurance
Focus: Detailed examination, finding issues

## Behavior
- Read code thoroughly before commenting
- Check security, performance, maintainability
- Verify tests exist and pass
- Ensure error handling is comprehensive

## Process
1. Understand the changes
2. Analyze impact
3. Check for issues
4. Suggest concrete fixes

## Tools to Favor
- Read for understanding
- Grep for cross-references
- Agent-based code review

## Output
Prioritized issue list with severity

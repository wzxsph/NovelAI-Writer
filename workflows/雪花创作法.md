# 雪花创作法工作流

## 触发条件
用户执行 `/novelai init --template snowflake` 或要求使用雪花创作法创建新项目。

## 输入
- 作品类型（都市/玄幻/穿越/科幻/历史）
- 一句话核心创意（1-2句话描述故事核心）
- 目标字数（30万/50万/100万+）

## 工作流程

### Step 1: 作品标签定义
**执行**: 询问用户选择作品标签

参考 @KB{作品标签} 提供选项：
- 题材标签（玄幻/都市/仙侠/科幻/历史）
- 类型标签（穿越/重生/系统流/种田流）
- 受众标签（男生/女生/通用）
- 情感标签（甜/虐/爽）

**输出**: 确定的3-5个标签

### Step 2: 金手指设计
**执行**: 根据作品类型，设计主角的独特优势

参考 @prompts/金手指生成.txt（如存在）

**问题**:
- 主角有什么独特能力/优势？
- 这个能力如何与主线冲突结合？
- 能力有哪些限制/代价？

**输出**: 金手指定义（一句话概括 + 详细描述）

### Step 3: 一句话梗概
**执行**: 将核心创意凝练为一句话

参考 @prompts/一句话梗概.txt（如存在）

**格式**: "[主角]因为[起因]不得不[行动]，最终[结果]"

**输出**: 一句话梗概（50字以内）

### Step 4: 一段话大纲
**执行**: 将一句话扩展为一段话大纲

**包含**:
- 世界观概述（1-2句）
- 主角定位（1句）
- 核心冲突（2-3句）
- 主线结局（1句）

**输出**: 一段话大纲（200字以内）

### Step 5: 故事大纲
**执行**: 设计故事的整体大纲

**结构**:
- 开头（1-2段）：引入世界观、主角、金手指
- 发展（3-5段）：主角遭遇挑战、成长、获得资源
- 高潮（1段）：最终决战/关键转折
- 结局（1段）：主角达成目标

**输出**: 故事大纲（500-1000字）

### Step 6: 世界观设定
**执行**: 根据作品类型创建世界观

参考 @KB{起名指南} 和 @KB{作品标签}

**包含**:
- 地理环境（主要场景、势力分布）
- 势力架构（主要势力、关系）
- 修炼体系（可选，如适用）
- 社会规则

**输出**: 世界观设定文档

### Step 7: 核心蓝图
**执行**: 设计核心蓝图

**结构**:
- 卷数规划（根据目标字数）
- 每卷核心事件
- 核心角色规划（主角+3-5个关键配角）
- 核心物品/设定

**输出**: 核心蓝图（见 @prompts/核心蓝图生成.txt）

## 输出汇总

工作流完成后，输出：
```json
{
  "作品标签": ["标签1", "标签2", "标签3"],
  "金手指": "描述",
  "一句话梗概": "xxx",
  "一段话大纲": "xxx",
  "故事大纲": "xxx",
  "世界观设定": "xxx",
  "核心蓝图": {
    "卷数": 5,
    "各卷标题": ["卷1", "卷2", ...]
  }
}
```

## 状态更新

将上述内容保存到 `.novelai_writer/state_document.json`：
- metadata.title = 作品标题（如有）
- metadata.genre = 作品标签
- metadata.word_count_target = 目标字数
- characters = 初始角色（如有）
- plot_threads = 初始情节线

## 后续步骤

雪花创作法完成后，建议：
1. 使用 `/novelai worldbuild` 深化世界观
2. 使用 `/novelai character add` 添加核心角色
3. 使用 `/novelai plan` 生成详细章节大纲

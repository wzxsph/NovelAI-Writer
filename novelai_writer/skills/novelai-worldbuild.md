# NovelAI Worldbuild

## When to Activate
- 用户要求构建世界观
- 项目初始化阶段
- 用户描述了一个新的小说设定想法

## Output Format

参考 `@prompts/世界观设定.txt`

生成的结构化内容：
- 世界观核心规则
- 修炼体系/能力等级
- 主要势力/组织
- 关键地点/场景
- 重要物品/道具

## 生成流程

1. 询问用户小说类型（修仙、都市、科幻、奇幻等）
2. 生成核心世界观规则
3. 生成修炼体系/能力等级
4. 生成主要势力/组织
5. 生成关键地点/场景
6. 生成重要物品/道具
7. 将设定保存到 `.novelai_writer/state_document.json` 的对应字段

## 示例

用户输入："我想写一个都市异能小说，主角有读心术"

输出：
- 世界观：现代都市，存在异能者隐藏于普通人中
- 修炼体系：觉醒等级 D-C-B-A-S
- 主要势力：异能协会、地下组织
- 关键地点：都市各区、异能者聚集地
- 重要物品：异能结晶、屏蔽器

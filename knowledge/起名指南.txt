# 起名指南

本文档提供小说创作中的命名规范，帮助作者创造有特色、有记忆点的名字。

## 命名分类

### 1. 人名

#### 姓氏模板

**玄幻/仙侠类**
- 古风单姓：凌、萧、云、墨、苏、白、江、柳、秦、欧阳（复姓）
- 神秘感：夜、幽、冥、寒、孤、傲、龙、凤
- 自然元素：风、雷、雨、雪、霜、云

**都市类**
- 常见姓：李、王、张、刘、陈、杨、赵、黄、周、吴
- 时尚感：苏、林、何、郭、罗、宋、叶、许、韩

**历史类**
- 大姓：萧、韩、柳、白、秦、李、王、张
- 古雅：裴、程、司马、公孙、慕容、尉迟

#### 名字模板

**单字名**
- 名门正派：浩然、子墨、子轩、承轩、锦年
- 邪派/配角：天邪、魔羽、幽冥、绝尘、孤影

**双字名**
- 均衡型：云霆、墨离、凌霄、霄汉、苍澜
- 柔美型：初晴、映雪、惜诺、灵犀、紫菱

#### 禁忌

- 避免过于常见的名字组合（李明、王芳）
- 避免谐音歧义（史珍香、杜子腾）
- 避免与现代人物重名

### 2. 功法/技能名

#### 命名规则

1. **属性+形态**：玄冰诀、烈火掌、疾风步
2. **意象+动作**：龙腾击、凤舞九天、雷霆一击
3. **数字+等级**：九转玄功、七星剑诀、三昧真火
4. **意境+效果**：天外飞仙、一剑隔世、万物化生

#### 风格词汇

**正道功法**
- 光明类：玄光、圣光、炎阳、光明
- 清净类：清心诀、静心咒、归元诀
- 仁德类：仁者无敌、厚德载物

**邪派功法**
- 阴森类：幽冥诀、鬼影功、噬魂大法
- 极端类：灭世魔功、屠神诀、破天裂
- 诡异类：妖神经、血魔变、蛊虫术

#### 禁忌词汇

```
终极、至尊、无敌、天下第ー、举世无双、唯我独尊
（除非是嘲讽或反讽）
```

### 3. 法宝/武器名

#### 命名规则

1. **形态+材质+类型**：玄铁重剑、紫金葫芦、玉如意
2. **意象+效果**：缚仙绳、捆龙索、天地镜
3. **历史+来历**：轩辕剑、盘古斧、女娲石

#### 分类词库

**武器**
- 剑：长剑、重剑、短剑、魔剑、神剑、仙剑
- 刀：大刀、弯刀、魔刀、血刀
- 其他：枪、戟、棍、锤、弓

**法宝**
- 防御：甲、盾、钟、塔、印
- 控制：铃、环、锁、网、阵
- 辅助：葫芦、扇、镜、珠、灯

### 4. 组织/门派名

#### 命名规则

1. **地名+流派**：青云宗、百花谷、玄天阁
2. **意象+门派**：万剑门、破天教、噬魂殿
3. **动物+称号**：天龙帮、虎啸堂、凤羽阁

#### 风格词汇

**正道**
- 清雅：青云、玄天、紫霄、玉虚、白云
- 正统：少林、武当、峨眉、昆仑、崆峒

**邪派**
- 阴森：幽冥、鬼域、魔宫、血煞、绝情
- 诡异：天魔、万毒、噬魂、九龙

**中立**
- 神秘：天机、玄机、神算、命运
- 江湖：天下、乾坤、江湖、侠义

### 5. 章节名

#### 命名模板

**格式**
- 4-7字为主
- 包含地点/人物/事件
- 有记忆点

**常用结构**
- 人物+动作：主角觉醒、主角出关、主角认主
- 地点+事件：宗门大比、天下大乱、群雄汇聚
- 悬念式：惊天秘密、身世之谜、命运转折

#### 禁忌

- 避免太直白（"第X章开始写"）
- 避免太文艺（"月下独酌"之类不适合网文）
- 避免剧透式（"主角死了"）

## 高风险词汇

以下词汇在网文中被过度使用，容易产生AI感：

```
系统、系统提示、系统面板、系统声音
叮、叮叮、恭喜宿主、获得奖励
量子、幽冥、幻影、终极、至尊、星穹
法则、规律、天道、命运、轮回、因果
```

## 命名原则

1. **可读性**：名字要容易读写，不要太生僻
2. **记忆点**：每个重要名字至少有一个记忆点
3. **一致性**：同类名字风格要统一
4. **寓意**：重要角色/物品的名字最好有寓意
5. **避免重复**：检查是否与知名作品重名
